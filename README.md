# Tasa_Oficial_BCV
Una API para obtener de la pagina oficial del Banco Central de Venezuela el tipo de cambio de varias monedas

# Uso
La API debe recibir un objeto JSON con las siglas de la moneda de la siguiente forma
```json
{"moneda":"USD"}
```
